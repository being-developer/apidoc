API DOCUMENTATION

Wingify -Product REST API

User API

  • /api/v1.0/register POST To register a user

    ParametersOptional(+)/Mandatory(*)format
    name*json/formData
    email*json/formData
    password*json/formData
    Info

    This endpoint of for registering the user.


    Return

    Json with message and success tags for example : {"message":"User Succesfully Created","success":true}

  • /api/v1.0/login POST To User login

    ParametersOptional(+)/Mandatory(*)format
    email*json/formData
    password*json/formData

    Info

    This endpoint of for logging in the user. Send the email and password in same format.


    Return

    Json with message ,success and token . if user is not valid token will be null for example : {"message":"Succesfully logged in","token":"399465c5867fe6802cac28ffb6f6cbf370c2d2e01e12a5c21aaf188fd34d","success":true}

  • /api/v1.0/logout GET/POST To logout

    ParametersOptional(+)/Mandatory(*)format
    accessToken*header

    Info

    This endpoint is for users to logout.


    Return

    Json of logged out with message and success for example : {"message":"successfully log out","success":true}.
