Horizon

  • /api/v1.0/login POST

    ParametersOptional(+)/Mandatory(*)format
    deviceId*header
    username*json/formData
    password*json/formData

    Info

    This endpoint of for logging in the user. Send the username and password in same format.


    Return

    access_token

  • /api/v1.0/respond_add_friend POST

    ParametersOptional(+)/Mandatory(*)format
    accessToken*header
    deviceId*header
    friend_id*json
    response(1,0,-1)*json

    Info

    This endpoint is for confirming friend or not allowing friend.

    Return
